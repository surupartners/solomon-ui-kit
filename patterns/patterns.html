3. Buttons

Normal sized buttons (this is standard sized text)

Buttons can be made from buttons or links:

Button made from <a class="button" />
Full width button
Button sizes
Large as link Small as link
Hollow button style

Use hollow buttons for significantly less important actions such as cancel, back, etc. Can also be used for somewhat destructive actions, but usually .button--alert is more appropriate.

Inline button style

Use an when a button should be part of normal text but where an anchor link is inappropriate (i.e. if the buttons triggers an action rather than navigating to another URL / resource)

Submittable buttons

Just add the classes js--is-submittable is-submittable. By default the Javascript application will bind the button's parent form submission and run the animation for you. If you'd like to handle this yourself simply omit js--is-submittable (leaving is-submittable for styling) and provide your own JS hook class such as js--submit-button. Obviously don't use js--is-submittable again!

Confirm buttons

Some actions such as deleting things should be confirmed before being applied. Use the markup found in the button and the Javascript application will automatically register listeners to make a confirm button.
